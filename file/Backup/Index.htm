Custom Design Software will back up  directly to Zip drives.  If you are using re-writeable CD's, you need to format them first using a product like Roxio's  Easy CD Creator.  When you click on the back up, select the drive you want to back up to.  Do Not use the C: Drive.  Follow the prompts.
