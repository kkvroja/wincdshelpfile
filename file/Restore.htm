Restore WinCDS Data

To restore your data from a previous backup, select this option from the System portion of the File menu.

NOTE: Restoring data will eliminate any transactions that you have created since the time of the backup you are restoring.

Select the portion of the Database that you wish to restore:

POS
Payables
Payroll
Banking
G/L
Store Setup
Pictures


Once the portion of the database is selected, select the drive and directory your backup is located in (for example, off a CD you have created). The list of relevant files should show in the box on the right, including the size and date.
