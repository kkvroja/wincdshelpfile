Open Website

Invoking this menu option will attempt to open your web-browser and navigate to your website. The website it uses will be the website you entered on the website generator page.

If you have not used this page yet, or you have not changed the default value, it will navigate to the demonstration website run by WinCDS at http://www.simplifiedposonline.com. This is a mock-up website that resembles a real Yahoo store, but with purchasing disabled.

Feel free to browse this site to get a feel for what can be generated for your store.
