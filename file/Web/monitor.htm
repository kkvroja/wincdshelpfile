Real-time Inventory Monitor

In order to automatically verify stock levels and process orders in real time from Yahoo Small Business Solutions, one copy of the WinCDS program must be running with this form open. This form performs two operations.

  1. Verifies stock levels
  2. Retrieves order emails

Verifying stock levels

The program opens a port to listen for a connection from the Yahoo server. Yahoo asks about a specific Style number and the program with the current stock, and a description if it is out of stock or non-determinable.

Retrieving order emails

The program connects to the specified POP3 server with the specified email account and password specified under the config and checks to see if there are any current orders. There isn't any action required. The program will continue to recheck at a particular interval of time so long as this form is open. If this form closes for any reason, simply re-open it, and it will begin its retrieval cycle again.

Configuration Panel

The first time you use this, you should set up the values in the configuration panel. Click the 'Config' button, and you will see the following screen:



Simply enter the appropriate values into the boxes.

Local IP Address

Can be obtained from your network administrator or through a website such as www.ipchicken.com.

RT Inventory Monitor Port

Any port number larger than 2048 that is not used by another application should be sufficient. Either use the default (the default for the program is 4242) or supply your own. If you have a network administrator for your local network, that person will probably want to specify this value.

Order Retreival POP3 Remote Host

You should use the specified host and IP number (pop.bizmail.yahoo.com 110) unless specified otherwise by Yahoo.

Order Retreival Email Account and Password

After setting up the email account using the Email Account Manager and designating your orders to be delievered to this account, enter the account and password into these boxes. Be sure to include the entire email address (e.g. orders@yourpersonalwebsite.com).

When you are finished with the configuration, click 'Save'.
