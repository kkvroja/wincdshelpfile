Steps to Establishing a website

It will take you apporoximately 2-3 hours to do your initial site setup.
You should be able to begin taking orders as soon as your Merchant Account is established.

Run WinCDS and select Generate Web Pages from the 'Web' Menu off the main form.
Select all options, and click 'Build'

Establish your items (You should only ever have to do this once!)
  • Goto Store Manager (Off of Services Page)
  • Select CATALOG MANAGER
  • Select 'Manage Your Tables'
  • 'Create New Table'
    1. Step 1: Enter 'items' for Table Name (and any description you want if you desire)
    2. Step 2: Select 'Next' to keep the mandatory fields
    3. Step 3: Select the following Store Fields: Caption, Abstract, Label
    4. Click 'Next'
    5. Step 4: Select the following Shopping Fields: Product-url, Manufacturer Click 'Next'
    6. Step 5: Click 'Add Another Custom Field' Add the following Custom Fields: 'Department', Format='Text', Required=No, Leave Default Value Blank Click 'Next'
    7. Step 6: Verify that you have 14 fields in your table. Click 'Save'


Now its time to put items into your Table (This should be done when changes are made to list of items (not changes in available quantity))
  • From the CATALOG MANAGER
  • Select 'Upload' or 'Upload Items'
  • Click the 'Upload' button
  • Select the destination table: 'items'
  • Choose Action: Rebuild
  • Click 'Browse' and select the items file (e.g. c:\inventpx\webtemplate\2data.csv)
  • Click the 'Upload' button
  • In the Warnings and Errors Box, You should see:
    "No warnings or errors to report. Please choose Commit to update your database."

  • Click 'Commit' to update the database.
  • Click 'Rebuild' to confirm the commit action.
  • The database is now built, but it is not 'live' yet (usable by the website).
  • To make it 'live', now click on 'Publishing', and then 'Publish Catalog'.
  • The button should gray-out and the Publish History should indicate that the catalog has been published.
  • Click on 'Store Manager' to return.


Now that you have your items installed, you can install the pages built in WinCDS.
Goto the Yahoo File Manager (once you're logged in, it is availble here:
      http://us.1.p.webhosting.yahoo.com/filemanager)

Select 'Upload Files'

You need to upload 2 files:
  • c:\inventpx\webtemplate\package.zip
  • c:\inventpx\webtemplate\templates\unpack.php
When you're done uploading those two files, return to the File Manager from the File Upload. You should see the two files in the directory now down below.

To the right of the file unpack.php should be a link entitled 'View'.
Click the 'View' on unpack.php and wait for the page to finish loading.
Your website, including all images and directories should now be unpacked into your directory.
To verify this, you can reload the file manager page and see all the files unpacked.
Additionally, you can now visit your store website: www..com (e.g. www.simplifiedposonline.com)


For this next step, you will need to create a new email account to process your order emails:
  • Goto EMAIL CONTROL PANEL From the Serivces Page
  • You should see that you have about 100 email accounts availble to create.
  • You can use this process to create as many as you want, although this will only walk you through the process required to create the accounts required by our system.
  • Click 'Create Address'
  • Enter the name for the mailbox (e.g. 'orders').
  • You need to create a mailbox, not a mail forward, so click 'Create Mailbox'
  • Because WinCDS will read the orders directly from Yahoo, select the option: 'POP Email Only mailbox'
  • Click 'Continue'
  • Enter the password for the mailbox in both boxes.
  • Enter your email address in the 'Send instructions by email' box.
  • Click the checkbox agreeing to the 'Terms of Service'.
  • Click 'Submit'
  • Read the summary information
  • Click 'Return to Manage Email Addresses'
  • View the information in the table below to verify the address was created successfully. Now that you've got your items and your web pages setup, you need to prepare for the ordering process:
  • Goto the STORE MANAGER
  • Under 'Order Settings', Go through each item and configure your site as you need to.
  • Under 'Order Forms', Read through each field and verify that all your settings are correct.
    • Espeically take note of 'Checkout URLs', 'Order Confirmation', 'Order Status',
    • Also take note that you can preview your order forms with the button on the left top and bottom of the page.
  • Under 'Order Emails' you will set up how to process your orders.
    • To use automated, real-time order processing under WinCDS, enter the email address for order processing under 'Automatic Order Processing'.
    • Select Send 'Immediately'
    • Select Format 'XML Format'.
    • Click 'Done'
  • Under 'Shipment and Order Status' you can control confirmation emails to the customer through Yahoo.
    • Edit fields as desired.
    • Click 'Update' When Done.
  • Under 'Configure Inventory', you will hook tell Yahoo what computer to ask to get available quantities of items in real time.
    • Start a copy of WinCDS and Open the Real-time Inventory Monitor.
    • On the 'Configure Inventory' Page, Select 'Real-Time Inventory' under 'Inventory Options'
    • Under 'Settings', determine your IP address (try visiting 'www.ipchicken.com' if you dont know yours)
    • Enter this value into 'Call Script':
      • http://<your-ip-address>:4242/InvCheck.html
      • For example, if your ip address is 111.222.333.444, enter: http://111.222.333.444:4242/InvCheck.html
  • Under 'Payment Methods' you set up how you want to handle money.
    • "Yahoo! Store requires merchants to have a merchant account in order to open for business."
    • You can either use Paymentech or an existing merchant account.
    • They are required to be a "FDMS Nashville platform-compatible merchant account provider".
    • You will need:
      • Name of Merchant Bank
      • Merchant Number
      • MID (Merchant Identification)
      • TID (Terminal Identification)
    • After your account is setup, select the cards they are authorized to process and that you wish to accept.
    • Click Update when you are finished.
  • Under 'Shipping Manager'
  • Under 'Tax Rates'
  • Under 'Foreign Orders', we suggest you Select 'US Only' Click 'Update' when you are finsihed.
  • Under 'Shipping & Tax Text', you can see how your shipping and tax settings will function.
  • When you are done, Return to STORE MANAGER and click 'Publish Order Settings' under 'Order Settings' This will make your settings live and the link should become plain text when it is done.
