Sales Staff

WinCDS's has the capability to track all of your employees and allow each of them access only the the portions of the software you desire to allow them to access-each with their own password.

Entering your Sales Staff



Enter each member of your sales staff:
  • Enter the employee's name
  • Enter an employee number for each employee. If this employee is not a sales agent, leave this field blank.
  • If the agent is paid commission, enter the commission percentage in the third field.
Be sure to click Apply to save any changes.

Password Setup

To change an employee's password, click the employee's name, followed by the Change Password button. Two password entry boxes will appear, and the Change Password button will change to Save Password. You do not need the employee's current password to use this function.

Change Permissions is used to alter the employee's group memberships. When this button is used, a selection screen will pop up listing the available groups, with the current memberships pre-selected. These groups determine which areas of the program the user's password will allow access to. See the User Groups section for more information.

User accounts can now be deactivated, rather than deleted. To deactivate an account, select the user's name and click Disable Account. To reactivate an account, check the Show Disabled Accounts option and select the user's name. The Disable Account button will be replaced with an Enable Account button.

User Groups

User Groups control access to various parts of the WinCDS program. Certain menu items, such as Store Setup, may require password authentication. Sales Staff can be members of many groups.

WinCDS Default Groups

WinCDS begins with several default user groups:
Group NameDescription
Administrators Have access to every part of the program
Office Staff Have access to the inventory control, ordering, and payments
Sales Staff Have access to sale creation, adjustment, and viewing sections
Warehouse StaffAre allowed to schedule and complete deliveries
Everybody * (See Below)


The Everybody Group

The Everybody group is used in conjunction with the Security Level setting found on the General Page of Store Set Up. Privileges granted to the Everybody group are handled differently when the computer is accessible to customers.
  • In Office Computer mode, program areas allowed to Everybody do not require a password to enter.
  • In Sales Floor mode, these functions will accept a password from any active user account.
This prevents customers from accessing sensitive store data.

Custom Groups

In addition, you may create Custom Groups. These may have any name, and any one-character abbreviation. No two groups may share an abbreviation, and they may only be deleted (not changed) after they have been saved. The default group names and abbreviations can not be changed.



Changing User Groups

Security Level

The final step is setting the Security level on each computer which is located on the General Tab of Store Set Up.



The Security Level option determines how often the program asks for passwords.
  • No Passwords disables all passwords checks.
    This is not recommended for active stores, as it leaves all the financial information visible to all users.
  • Sales Floor mode assumes the computer is in an area accessible to customers, and will require passwords for most functions.
  • Office mode assumes the computer is in a physically secure environment, where only trusted personnel can use it. In this mode, non-sensitive functions do not require passwords.

The program areas that Sales Floor and Office modes protect are configurable through the User Groups screen. These modes only affect areas permitted to the Everybody group.

Click Apply to save the Security Level for each store.
