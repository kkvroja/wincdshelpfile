Security Level

The Security Level option determines how often the program asks for passwords.

Select Demo mode to disable all passwords checks.  This is not recommended for active stores, as it leaves all the financial information visible to all users.

Sales Floor mode assumes the computer is in an area accessible to customers, and will require passwords for most functions.

Office mode assumes the computer is in a physically secure environment, where only trusted personnel can use it.  In this mode, non-sensitive functions do not require passwords.

The program areas that Sales Floor and Office modes protect are configurable through the User Groups screen.  These modes only affect areas permitted to the Everybody group.
