Multi Sales Tax

If you deliver to other areas which have a different Sales Tax Rate, you need to use the Variable Sale Tax.  This will be preset up for you.  If you need to add additional rates, this is the procedure. 

To add a rate to the Multi Sales Tax rates list, follow these steps:
  1. Click on the drop down Multi Sales Tax. The main default Sales Tax Rate must be listed first in the Multi Sales Tax List.
  2. In the empty box below labeled, Sales Tax + State, enter the new tax rate in the following format
    .xxx NAME
    For example
    .075 OH
    (You MUST HAVE 1 blank space between the rate and the area).
    It is recommended that you use a 2 digit abbreviation for the tax area.

  3. When you have entered the rate in the correct format, press the TAB key to add the rate.
For example,
.06 MI(default)
.075 OH(New Rate)
Be sure to click the TAB key to set the tax into the Multi Sales Tax List.  To add another tax rate just repeat the steps. When complete, be sure to the Apply Button to save the sales tax list.  This must be done on the server.

Note: The C button clears the variable sales tax list.
