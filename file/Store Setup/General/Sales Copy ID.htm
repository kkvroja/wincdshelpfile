Sales Copy ID

In this drop down, the names of the Bill of Sale headings will appear. The defaults are:

  • File Copy
  • Customer Copy
  • Delivery Copy
  • Salesman Copy


They can be changed, if necessary. To change, select the slot of the Sales Copy ID, 1 through 4, and then select the copy type from the drop-down beneath. Tab off the drop-down to set the copy. Be sure to click Apply to save your changes.

Note: The customer copy does not show the manufacturer and style numbers so you can't be shopped by unscrupulous customers.
