Advertising Types

How the Advertising Types Work in the software

You can personally configure the Advertising Types in our software.

To use this feature, the salesperson must simply select from the advertisting drop down. When a new sale is written, the salesperson can enter the information in the box on the sales slip to answer the question: "How did the Customer Come Into the Store?". This is all the salesperson must do in addition to creating a normal sale.

Then, at the end of the month, or when ever you choose, you can run the Advertising Reports to get your Advertising statistics.

Setting up your Advertising Types

  1. To enter the Advertising types, click on the drop down labeled Advertising.
  2. Select the type to change (e.g. Type 1) so it turns blue.
  3. Then in the empty box labeled Enter Advertising Types, type in the correct type, for example, Radio.
  4. Hit the tab key to make the replacement.
  5. Do the same for the balance of the advertising types.
  6. When complete, click on Apply on the form to save the changes.
You must do this for all stores in the chain. Be sure to save the changes in each store.
When you are finished you can click Menu to exit the setup screen.
