Your store's name and address information will be preset up here. After making any changes, Click Apply to Save the change.

If you use a store logo, you still must have this set up for other modules to tie into.

Store Logo

To install a store logo, place a file of type jpg, bmp, or gif named as follows:

StoreNLogo.xxx
where N is the store number and xxx is the appropriate extension.

The required dimensions are approximately a business card (approximate aspect ratio expected: 2.88 x 1).

For Example, a good dimension for your logo file would be 1300x450. This should ensure decent picture quality when printing your sales slips.
