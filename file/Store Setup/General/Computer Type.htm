Computer Type



In the upper right corner of the General Page, you can select whether this computer is the Server or is a Workstation. They will be preconfigured on the original CD's.

If a workstation is unable to find the server, a prompt will come up saying that it cannot find the Server. Is this the Server?

If a computer is a workstation, please be careful to answer NO to this prompt to avoid complications.

In the event that it is accidentally changed to a server, simply click on work station and save the change.
