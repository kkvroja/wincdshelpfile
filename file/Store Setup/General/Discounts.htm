Discounts

You can personally configure the Standard Discounts for making sales in our software.

When a new sale is written, the salesperson can click on the Dollar Sign () and then either enter a manual discount or select from a list you have setup from the Store Setup's General tab.

Create a Discount

To setup a discount,
  • Click on the drop down labled Discounts.
  • Select the empty item at the top of the list.
  • If the dialog does not come up immediately, select the box below labeled Enter Discount Types.

You should now see the following discount screen:    

Enter the information as well as a title for this discount (e.g. Truckload Sale, Overstock Clearance), the type of discount, and press OK. You can also click Cancel.

Edit/Delete a Discount

To edit or delete an item, select if from the list as above and simply edit the the information in the same box as above. Either click OK or Delete as desired.


Saving Changes

The Discounts drop-down will update to reflect your changes.
There is no need to click Apply on the Setup Form after making this change as it is done immediately.
