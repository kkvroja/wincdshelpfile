Factory Numbers



If you do not want customer to shop your prices, you have the ability to disguise your manufactures style numbers on the retail price tags so the customers cannot take the tags to your competitors.  Some stores already use a private store cost code to put the cost of the item on the retail price tags.  If you don't have a store cost code, choose a phrase or words that is easy to visualize in your head.  A suggested code is:  B U Y  F O R  C A S H.  The B = 1, U = 2, Y = 3 and so on.  If you click on Use Store Code, you can select to have the computer transpose the Factory Style Numbers into your private code.  You also can have the Cost, in code printed on the tags. 

To activate this feature, click on Use Store Code, and either or both Style Numbers in Code and Cost in Code.
