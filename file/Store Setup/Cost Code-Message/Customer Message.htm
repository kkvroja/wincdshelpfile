Bill of Sale Customer Message



There are three places in the software that you can have customer messages.

  • On Bill Of Sale, you can have a customer message with your store own terms and Conditions.  The default is:

Special Orders Are Not Subject To Cancellation!   

Received in good condition: 

____________________________________

  • If you are using the POS Epson Printer with the Cash Register interface you can have a customer message.  The default is:

Thanks for shopping with us.

      Come again soon!

  • If you need additional space for your customer terms, Clicking on Store Terms will allows you to have a much larger customer message.  This message appears on the customer's receipt on the left side replacing the Manufacturer and Style number columns.  You can design this area completely in WordPad.

To change the default messages, Double Click the current message block on the screen.  Compose your message and click the Apply button.
