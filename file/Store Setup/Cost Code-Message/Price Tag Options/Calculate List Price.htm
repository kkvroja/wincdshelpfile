Calculate List Price

There are three ways to have the computer calculate you Factory List Prices for retail price tags on the floor. 

 

These options are used to tell the computer how to calculate the Factory List Prices on the Retail Price Tags that you print out.  All the calculations themselves are done under  Inventory, New Items. 

  • The default,  in the Drop Down in Store Setup is 2 x Landed + Percentage.  In the factory List Add box under Inventory, New Items  you can enter a number like 40.  For this option, the computer will take the Landed Cost times 2 plus 140%.

Example: Based on $100.00 Landed Cost x 2 = 200.  Plus 140% = $280.00 If don't Merchandise Prices was checked on, these prices would be actual.  If Merchandise Prices was not checked, the list price would back down to $279.95.  The same would be true for the On Sale price

  • If you checked off On Sale x Entry, the calculations would be as follows:  Entering 1.4 in the List Add box would render the list price as follows:

                Example:  Based on Sale price of $200.00 x 140% = $280.00.  Enter 1.4 in the List Add box. 

  • If you checked off Round to Dollar, the software uses the next whole dollar.

               Example:  If the On Sale price turned out to be $266.56, the software would change the price to $267.00.
