Integrated Credit Card Processing

In this section, you can select and configure Credit Card swiping for this computer.
First, you must have an account through one of our Credit Card providers.

  1. X-Charge
  2. TransFirst, LLC

Select the Credit Card processor you want from the drop down. Additionally, ON EACH WORKSTATION, after you have set up the server, you must click the "Use CC Machine" checkbox if this computer will have a physical Credit Card swiper attached. This must be done on a per-workstation basis, as not computers will always have a credit card swiper attached.

After you have selected your company, the box will change to one of the options.

X-Charge

Here, all you can enter is your Merchant ID, and an optional Merchant ID for a MOTO account for this store. To set up transction directories, if the default values are not sufficient, please use the Credit Card Management panel.

Transaction Central (TransFirst, LLC)

You must enter your merchant ID (TCID) and RegKey. Additionally, if you do not want to support American Express or Discover, make sure these boxes are cleared.
