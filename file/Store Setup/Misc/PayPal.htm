PayPal Setup



If you need to integrate with PayPal to correlate online sales, you will need an authorization code from WinCDS.

After you have entered your PayPal Authorization code from WinCDS, you will be allowed to enter your PayPal credentials for integration.

PayPal Credenitals



When you are ready to enter your PayPal information, you will need first to acquire the information from the PayPal Website.

Follow these instructions to obtain your PayPal API information.

PayPal Setup

  1. Log in to your PayPal account.
  2. Click the My Account tab.
  3. Click the Profile tab. If you haven't already done so, you need to verify your account before requesting API credentials.
  4. Click Request API credentials under Account information (or, click My Selling tools on the left, and then API Access).
  5. Select Option 2, request API credentials.
  6. Generate API username, password, and signature (these are different than your regular Paypal credentials).
  7. Copy and paste these into the fields in the WinCDS setup form under store setup.


NOTE: If you are going to enter the information, be sure to enter ALL the information, or leave it all blank. Otherwise, you will be told that the login information is failing every time that WinCDS tries to connect to PayPal.
