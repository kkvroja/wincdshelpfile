Use Cost on Shipments

Use this option to specify how you want costs handled. Select the cost-handling method you wish to use from the combo box and be sure to click Apply to save the setting.

The available options are:

TypeDescription
Use CurrentUse the current cost always - never update the price.
LIFOUse the 'Last In First Out' method for tracking item prices.
This means the price used will be the price of the most recent item added to inventory.
FIFOUse the 'First In First Out' method for tracking item prices.
This means the price used will be the price of the item in the inventory the longest.
Average CostUse the current cost always and update it to the average item cost when receiving a new PO.
If you want to use Inventory Cost Averaging as shipments arrive, use this function to set up averaging.
Manual OnlySelect this option if you wish to manage your own costs. WinCDS will never change the cost of an item automatically with this option selected.
