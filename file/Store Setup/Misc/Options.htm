Options



Click Below for Explanation

Sale Options

Manual Entered Bill of Sales Numbers When you are first set up the software, you will want to use manually enter Bill Of Sale numbers to duplicate the open orders you already have. Be sure to click on the Apply Button to save changes. After you have entered all previous sales, go back and reset to have the computer generate the sale number. You can select the starting automatically generated Bill Of Sale Number.

Note: You can set the software to increment the sale number from where you manual numbers left off. However, I would start from an even number some distance for the manual sales.
Require Advertising TypeIf you want to force your sales people to select how the customer came into the store when they are making a sale, select this option and Apply.
Prompt Email AddressIf you want force a prompt for an email address when creating a sale, select this option.
Prompt For Address on Cash RegisterIf you want the Cash Register interface to ask for an address for the customer, select this option.

Multi-Store Options

Multi Store: Sell from Login Location If you have multiple stores and use a central warehouse, do not check this option. Not checking this will set the default location of Inventory in the Point Of Sale module, to the store you are logged into. If you want the merchandise to be pulled from the central warehouse, check this option.

This will eliminate your salesperson forgetting to select where the merchandise is when making a sale.
Multi Store: Post POs & Bank to Loc 1If you have multiple stores and want to pay your invoices out of the Main Store's checkbook, click on this option and save. This will send the cash amount of the Purchase Orders directly to store 1 checkbook.
Multi Store: PO Invoices to Loc 1When you check this option, all PO invoices will be sent to store 1, irregardless of which Location the PO is written for.
Multi Store: Lock Tag PO to Login StoreWhen creating a new sale, checking this option will lock the Tag Incoming Stock option under selecting a style number to the store you are currently logged into.
Multi Store: One Calendar for All StoresClick this option if you want the delivery calendar for this store to show the deliveries for all stores by default.

Programs Options

Use Delivery Time WindowsCheck this option to allow recording a delivery time window in addition to a delivery date for sales.
Use Wireless Base UnitCheck this option if you are using a wireless base unit (e.g. 8061, 8071-C) to allow you to communicate with your hand-held scanner. Checking this option also allows you to access the Configure Wireless Scanner menu option off of the main File menu.
Start Menu MaximizedChecking this option will cause the main WinCDS form to start in full-screen mode as opposed to the normal, windowed mode. This is useful for stores that are set up to run as a cash-register.

Other Options

Labor is Taxable If Labor Charges are taxable in your state, Click this option and Apply to save.

Available Options:
  • Never - LAB charges are Never Taxable.
  • With Sale - Tax LAB charges only when the sale is taxed. If there is no TAX charged on the sale, LAB charges will be considered non-taxable as well. If there is TAX charged on the sale, LAB charges will be taxable.
  • Always - LAB Charges are Always Taxable.
Delivery is Taxable If Delivery Charges are taxable in your state, Click this option and Apply to save.

Available Options:
  • Never - LAB charges are Never Taxable.
  • With Sale - Tax LAB charges only when the sale is taxed. If there is no TAX charged on the sale, DEL charges will be considered non-taxable as well. If there is TAX charged on the sale, DEL charges will be taxable.
  • Always - LAB Charges are Always Taxable.
Default Delivery PercentIf you specify a percentage in this box, this will be the default percetage to apply to delivery charges (DEL) when creating a new sale. The value can be edited afterwards, but this will allow you to set the default rate.
Receiving Labels Specify this option to determine which receiving labels will be printed.

Available Options:
  • None - Print NO receiving labels.
  • All - Print receiving labels for ALL items.
  • Just Stock - Print receivinng labels for JUST STOCK.
Use Cost on Shipments If Delivery Charges are taxable in your state, Click this option and Apply to save.

Available Options:
  • Use Current - Use only the latest item received to determine new item cost.
  • FIFO - Basic Item Queue, First-In-First Out.
  • LIFO - Basic Item List, Last-In-First-Out.
  • Average Only - Uses Average Cost of all items in the system at time of receiving.
  • Manual Only - Does not change prices automaticaly at all. Prices will only be changed manually when the store owner chooses.

PO Options

Post POs to Accounts PayableEnables or disables the automatic posting of POs to Accounts Payable. After the Accounts Payable module is up and running, you can have the cash amount of the Purchase Order automatically posted to Accounts Payable when this option is activated.
PO Special Instructions Checked OnRemoves the entire Special Instructions box on the PO form.
Print PO w/o CostIf you prefer to send in Purchase Orders with No Cost, select this option and Apply.
Post Deposits to Bank ManagerTurn this on to coordinate WinCDS with the optional Bank Account Manager.

PO Special Instructions

Change the Special Instructions available on the PO form. Use the Up and Down arrows to select 1 through 4 and change the text. Leaving them blank will revert them to the default text.
