Cash Drawer

If you are using our Steel Cash Drawer, you need to select the COM Port to make the Drawer open when you tender cash, or indicate it is a USB Cash Drawer if so equipped..

If you do not know your COM port, you can usually find it quickest just by trying them until you find which one works. Start with COM Port 1 and move up.

Alternatively, if you know how to find and use Window's Device Manager, it will list the COM ports towards the bottom of the list under Ports (COM & LPT). To open the Device Manager, hold down the Window Key and press the Pause Key (upper right of the keyboard). If you are using Vista or higher, there will be an Administrator link on the right to open the Device Manager. If you are using Windows XP, click on the Hardware tab, and then the Device Manager button.

USB Cash Drawer

If you are using the USB Cash Drawer, use the up and down arrows until the box says USB. The software will automatically detect and operate your Cash Drawer with this setting.

Testing your Cash Drawer

To test your cash drawer, click Test and it will manually open the drawer.

Click on Apply to save each time.
