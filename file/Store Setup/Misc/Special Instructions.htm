Special Instructions

In the Special Instructions box, you can set the default messages that will appear in on your Purchase Orders under the PO system. If you leave the first three blank, they will be filled in with appropriate default values, and the last one is customizable on the PO form (although a default message may be supplied for it as well).

Use the up and down arrow to select the message to specify (as indicated by the numeral to the right of the box) and enter your new message into the box.

Tips:

  • To revert to the default message, leave the box empty.
  • To leave the box blank, enter a single space.
