Command List - Speech Recognition

Below is a list of the commands recognized by the WinCDS Speech Recognition engine.
"Click (Button Name)"Clicks the button identified. For example, "Click OK" would click the OK button.
"Tab" or "Next Field" or "Go Forward"Goes to the next field. This would be the same as hitting the TAB key.
"Go Back" or "Previous Field"Goes to the previous field. This would be the same as hitting Shift-TAB.
"Clear"Clears the current contents of a Textbox.
"Backspace"Performs the same operation as hitting the Backspace key.
"Up" or "Down"Moves up or down in a List.
"Select"In a list, selects the current item.
"Left" or "Right"Fix part of a word. Moves the cursor within a Textbox.
"Spell It"Switches from Dictation Mode to Spelling Mode to spell something out, letter-by-letter.
"Correction"Highlights the current contents of a box so that your next command will overwrite the text that is already there."Dictate"Goes back to Dictation Mode, allowing the computer to try to figure out whate you are saying.
(Field Name)For forms with many fields, you can simply say the all or part of the name of the field you wish to jump to.
"Voice Off"Turns off speech recognition.
