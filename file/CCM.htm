Credit Card Manager

From this form, you can configure and perform maintenance regarding your X-Charge Credit Card processing.

Options

Click Options to open an X-Charge panel to edit X-Charge options.



X-Charge Settings

Click Settings to change options such as the Transaction Folder for WinCDS to communicate to X-Charge.

The current Transaction Folder(s) that will be used displayed in the bottom right of this panel. Whether you use the default WinCDS Transaction Folders or specify your own through this panel, the current folder for the store selected in the top right of this form will always be displayed at the bottom.

Your X-Charge Transaction Folder

By default, WinCDS uses your server's CDSData directory. If this computer is the server, your transaction folder will be automatically configured to be locaed in C:\CDSData\X-Charge\LocalTran.

MOTO Account

Some dealers use a MOTO account for all manually entered credit card transactions. If you need to have a MOTO account, you can do this automatically by simply creating a directory named C:\CDSData\X-Charge\LocalTran-MOTO. If you have a MOTO directory set up, the software will automatically detect this and send all manually swiped transactions there instead of the normal LocalTran folder.

Individual Store Transaction Folders

If you require that each store use a different merchant account, and hence use separate Transaction Folders, WinCDS can automatically handle this. Simply create a folder for each individual store. For example, if you have three (3) stores, each needing its own directory, simply create, inside of the C:\CDSData\X-Charge folder, the following directories:
  • C:\CDSData\X-Charge\LocalTran1
  • C:\CDSData\X-Charge\LocalTran2
  • C:\CDSData\X-Charge\LocalTran3
  • C:\CDSData\X-Charge\LocalTran-MOTO (if a MOTO account is required)

Manually Specified Transaction Folders

If you choose to, you can also set folders PER COMPUTER. You can specify them here and they will be used instead of the WinCDS defaults, but this will be necessary to set on each computer as this will be stored in your system registry, and not in the WinCDS configuration files. You can either set a Transaction Folder for all locations or for specific stores. You can also set a Transaction Folder for All Locations, and then specify a folder for one of the stores. If no specific store entry is specified, if will use the setting for All Stores. If these are not set, the software will use the default WinCDS settings.
Verifying Transaction Folder Settings
To verify your Transaction Folder settings, simply click the up and down arrows in the top right of this form to change store number. Simply check the Transaction Folder(s) in use on the bottom to make sure they are what you want.

Transaction Central Settings

There are no settings for Transaction Central on the CC Admin Panel.



Returns

Be sure to enter the amount for all return transactions from this screen. If you fail to do so, you will be prompted for the amount.

From this form, you can perform the following operations:

OperationDescription
Return CC PurchaseThis allows you to return a Credit Card Transation manually, if necessary.
Return Debit PurchaseThis allows you to make a Debit Card Return Transaction manually if you need to.
Return Gift Card PurchaseIf you have the swiping of Gift Cards available, you can return Gift cards through this function, if necessary.
Obtain ReportThis allows you to query X-Charge to get a report of Credit Card transactions run in the past day.

Note: Only the transactions types supported by your configuration will be available. Otherwise, they will not show up in the list.
