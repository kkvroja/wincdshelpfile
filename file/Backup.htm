Backup WinCDS Data

You should backup your WinCDS every day!

The options for backup are:

POS
Payables
Banking
Banking
G/L
Everything


Custom Design Software will back up directly to Zip drives. If you are using re-writeable CD's, you need to format them first using a product like Roxio's Easy CD Creator. When you click on the back up, select the drive you want to back up to.

It is strongly recommended that you do not use your C:\ drive for backups, as this defeats the whole purpose of making a backup in the event your hard-disk fails.
