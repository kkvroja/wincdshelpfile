Email Config Panel

You currently have three options for sending email.
  • Microsoft Outlook or Outlook Express
  • via SMTP Host
  • Direct

Aquiring SMTP Host Information - Email Config Panel

If you are sending via an SMTP host, you will first need to aquire your host address and authentication. If you are going through Outlook or Outlook Express, you can skip this step.

This will explain how to aquire your SMTP host information from a computer configured to use Outlook or Outlook Express from Microsoft.

Aquiring SMTP Host information from Microsoft Outlook

From the main Outlook screen, Click the Tools menu and select Email Accounts...



If the wizard comes up with the Add a new e-mail account option selected, you do not have any email accounts set up on this computer with Outlook and you cannot proceed.

Select the View or change existing e-mail accounts and click Next >. Select one of the email accounts, and click Change....



From this screen, you can record your SMTP Host information. Record the following information and enter it into your WinCDS Email Configuration Panel from the File Menu:

  • Outgoing mail server (SMTP)
  • Username
  • Password (you will need to supply this yourself, as it will be encrypted on this form).

Aquiring SMTP Host information from Microsoft Outlook Express

From the main Outlook Express screen, go to the Tools menu and select Accounts.... From the dialog, select the Mail tab. Select your account and click Properties. Select the Servers tab.





From this screen, you can record your SMTP Host information. Record the Outgoing mail (SMTP) server name. If the checkbox My server requires authentication is checked, then click the settings and record the settings on the form the appears (either the user name and password, or use the same as the incoming mail settings).

Configuring your Email

Once you have gotten your email settings from your email program, you can enter them into WinCDS. Below shows how to enter the SMTP information (if required) and finish configuring your email.

Sending through Microsoft Outlook


To send email through your current Microsoft Outlook settings, select this default option.

You must have either Outlook or Outlook Express set up on the local machine (the one where the software is running). Otherwise, Outlook will receive the email to send, but it will remain in the outbox and no go anywhere.

If you are using Terminal Services, it will use the email settings from the server.

Sending through SMTP Host


You can send through the SMTP host provided by your ISP. You will need to know your SMTP Host (and port, if other than 25). You will also need to know your username and password.

For help determining your SMTP Host if you are already using Outlook or Outlook Express, click see the instructions here.
