Print All Tags



You can use this panel to print all the price tags for your inventory.

You can either print all or a sub-set of your inventory tags by selecting manufacturer and/or department. The number of tags displayed will show up indicating how many price tags the currenct selection would generate so you can plan ahead before you print.

You can select to print all the tags based on the following methods:
  • Current Store Quantities
  • Based on On-Hand quantities
  • Based on Available quantities
  • 1 per style number


Note that with the exception of the last item, if there is no quantities for a given item, no price tag for that style number will be printed.

Be sure also to select the tag size you desire before printing.

You can also select which printer will be used by clicking Setup.

Click Close when done.
