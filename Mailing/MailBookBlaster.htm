Email Blaster - Mailing Address Book



Email Blaster

Here you can send out a mass email to selected customers. After selecting the options and filters you wanted in the Address Book, entering this form will allow you to send a pre-prepared HTML email to all of your selected customers.

Creating an HTML email

The email blaster takes an HTML-formatted message. This can be created by most word processing software, including Microsoft Word, Open Office Write, and Microsoft Front Page. Generally, you can select Save As in the program you are using and select the file type at the bottom of the dialog box. Select as HTML or Web Page and save the document. You can then include it into WinCDS.

Settings

This will use the same email settings you set up in the Email PO interface panel.

Sending

To send an email, simply edit the subject for your mail and then click Browse... to select your file. When you are ready, click Send.

Details

To view the details of your email session, you can click the Details button to view the log.

Exiting

To exit, click Cancel.
