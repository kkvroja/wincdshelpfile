Mailing Address Book



Viewing all addresses

Here you can view all or a subset of your addresses. To refresh the list with all the addresses, click the Show All button.

Searching/Viewing a subset of addresses

To view a smaller selection of addresses, type in some text you wish to search for in the Search For box and click Search.

For more powerful searches, you can put a ^ at the beginning of the text or a $ at the end of it to indicate that you wish to search for those instances where your text is at the beginning or end of a field. That is, if you wish to find all of the zip codes that start with 56, type "^56" in the search field, select Zip from the In box and click Search. Or, if you wish to search for all of the Cities that end in NJ, and not just contain the letters "NJ", type "NJ$" into the box. This will cause the search to only select those that end in this text.

Printing the current list of addresess

You can print the currently displayed list of addresses by clicking on Print Addresses.
NOTE: It is always a good idea to look at the status bar at the bottom of the screen to see the number of records displayed to get an idea of how many pages this will produce.

Exporting the displayed addresses

After selecting this option, you will be asked whether you want a simple export or a more complicated one.

Selecting the simple export will send the displayed addresses to a text file that will be separated by commas (or a comma separated values type file, or a .CSV). This commonly accepted format will allow you to do a mail merge or import into nearly any other program you wish to use.

There is also an option to export only the selected records using the Export Mail form. This will not, however, export any fields other than the standard mailing fields.

Editing an address

To edit an address, simply double click on the line and you will be brought to the mailing add/edit section.

Other Options...

If you wish to see other information about your mailing list such as the amount that has been sold to customers, change the sorting, or to limit a list to the first 100 or so items, click on Other Options... and a side panel will appear.


Customer Sales History

You can display the Cost of items sold, the selling price, or the approximate Gross Margin for each of your customers. You can also optionally set a from or a to date for these calculations. Simply select the columns you wish to add and set the dates accordingly. When you are done, click on the Update button in the lower right.

Sorting and Counting

To limit the number of records returned or to change the default sorting method, select the desired values from these two combo boxes. To refresh your table, click Update in the lower right.
Note that all of the various fields work together. If you still have items in the Search For field, these will be figured into your search as well. Also, the options you select under the List Management panel will stay in effect even if it is no longer visible.

Exiting the Address Book

To exit the address book, click on Menu.
