Print Mailing Labels

To print mailing labels for the software, click Mailing, Print Mailing Labels. A "Select Sort Method" form will appear. At the bottom of the form, it shows the quantity of labels required. Divide this number by 30 to get the number of sheets of labels necessary. Choose to print sorted by Zip code for Presort or Bulk or alphabetically for first class.   You can select to add  the customer telephone number, message saying "Or Current Resident" or no message. Click Print.

Use any of the following labels

  • Avery 5260

  • Office Max 86101

  • Any standard Laser 3-Accross, 10-Down labels
