Export Mailing List

If you use an outside mailing service, click Mailing | Export Mailing List.



  1. First, select the store to export.
  2. Next, select the format to export to. Currently supported formats are:
    • Text -- uses the standard CSV (comma-separated values) format.
    • Microsoft Excel
    • Microsoft Word
    • Microsoft Word Mail Merge
    • Email List (text) -- A list of just the entered email addresses.
  3. Finally, enter the file you want to export to. You can use the Browse button to find the drive and directory you want easier. Click Export to begin the process.

Your mailing list will be exported into the specified format.

A Note on using the Microsoft Mail Merge

To use the Mail Merge feature, you need to select an already created merge-ready document. To create one, select New... from the File menu inside Microsoft Word. Select Templates On my computer.... Under the tab labeled Mail Merge, select the type of document you wish to create. Fill out the letter template and save it in a location you can find. When you have to select the file for output on the Export screen, select your template. It will open that up and use it to create a letter for each customer. When it is done, it will leave a document open on your computer with one letter for each customer in your list. With it, you can either save, discard, or print it from within Microsoft Word.
