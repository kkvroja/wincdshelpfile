Print Preview

You will see a Print Preview button for most reports.
Note: Some columns may be slightly misaligned on screen, but they will print correctly on a printer.

The Title Bar (at the top of the screen) will show the name of the report, current page, and total pages.

Navigating the Preview Document


Use the control buttons in the lower-right corner of the Print Preview screen (or keyboard shortcuts) to navigate:

Button Keyboard Shortcut Description
Print Page P A dialog box will ask you what and how to print. You can select All or a page or a single page range (e.g. 6-11).
Goto Page G or * A dialog box will ask you what page you wish to scroll to
Up Up Arrow or Right-Click Scroll the page up
Dn Down Arrow or Left-Click Scroll the page down
Pg Up PgUp Scroll the report up one page
Pg Dn PgDn Scroll the report up one page
Home Home Top of Document
End End Bottom of Document
Done ESC Exit

Using the Mouse to navigate

You can also use the mouse to navigate.
  • Left-clicking on the page is the same as Page Down
  • Right-clicking is the same as Page Up.
