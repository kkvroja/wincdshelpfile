Entering Open Sales

The software defaults to automatically generate the Bill of Sale numbers. Temporarily disable this function by going to File: Store Setup: click the Misc tab, then check the Manual Entered Bill of Sales Numbers. This will allow you to use your original Bill of Sale numbers for the open sales. Click OK to save this change.
Note: You must exit and restart the software for this change to take effect.

See Create a new sale to add open sales to the system.
Tip: Use the Date Slider in the upper-right corner to set the original sale date.
Note: Enter all open sales that are not delivered when doing this, including Special Orders, Lay A Ways, sales to be Delivered, etc.

If you want the computer to automatically generate the Bill of Sale numbers, don't forget to re-enable this function (see top of page).
