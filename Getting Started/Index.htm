Getting Started

How to Get Started with Custom Design Software

Note: All sales must go through the system twice. Once at POS (to create the sale), and a second time when the sale is delivered (to take the item out of inventory).

Getting Started with 'Custom Design Software for Furniture Stores' is a 3-step process

  1. Loading Original Inventory
  2. Entering Open Sales
  3. Entering Existing Installment Contracts (If applicable. Installment Module Required.)

How-To Topics

  • Create a new sale
  • Add new inventory
  • Check item details, price, inventory allocation, and print tickets
  • Create a Package Kit
  • View deliveries scheduled for next 31 days
  • Pay commissions
  • Change commissions after the sale has been written
  • Check status of an order
  • Password Setup
  • Balance Your Cash Drawer
  • Set up and Installment Financing Sale

You can setup a delivery when Creating a new sale, or after the sale is created through Check Order Status.
