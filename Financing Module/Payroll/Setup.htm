Setup

Open the Payroll module for the proper store by entering L1 for Location 1 L2 for Location 2, etc.  Enter the User ID and password  (You must call Custom Design Software to get this code). 

To set up the Payroll Module, you will need to get all Federal, State, City and any other local tax tables.  These are usually mailed to you directly each year.  We try to maintain the Federal Tax tables but it is recommended that you review the tax tables yourself. 

Adding Tax Tables

Using the Michigan tax tables for reference, add your own tax tables for State, City and local taxes.  Remember, use the ANNUAL chart.  The computer backs the annual rate down to correct pay period. 

To build a new State Tax Table, for example, Click on Administration; Tax Tables.  Click on Add and you have a blank table screen. 

First, use a 3 digit code to represent the tax table.  For example MIU would stand for Michigan Unemployment.  Next type in the full description.  Then the abbreviated description that will be printed on the checks.  Mi SUTA would stand for Michigan State Unemployment Tax.  Fill out the table or enter a percentage as your tax requires.  Please don't forget to click on the correct dot on the right side for State unemployment or State Withholding as the case may be.  When you finished with the table, click Add.  This sets the table in the database.  Finally, if there are no more tables to add, click Stop Add to finish.

 

Adding Employees

To add your employees, click on Administration; Employee Maintenance.  Click Add to start entering employee.  For the Employee Code, use up to 8 characters to abbreviate the employees name.  Do NOT use their employee number as in the sample below.  Enter the employees data as indicated.  Be sure to be exact as the software will print out W-2's at the end of the year.  Select Shift 1 from the drop down.  Frequency is how often you pay the employee.  Type is Salary, Hourly etc.  Rate is extremely important for the software to work correctly.  If the employee is Salary, then put in the ANNUAL SALARY not the weekly salary.  For Standard hours, put in 40.

 

 

Tax Code Button

If you have any additional taxes to deduct from each employee, click on the Tax Codes button.  The following blank screen appears.  This screen may not be used by all employees if they don't have any additional taxed to deduct.  On the first line, enter the abbreviation for the tax to be deducted.  In this case, Michigan Withholding.  Fill in the Liabilities and expenses as indicated.  Notice that the State Unemployment is listed a the bottom.  Click the Ok when complete.

 

Special Deductions

If the employee has any "Special Deductions" such as Medical Insurance, Profit Sharing, or Child support have to be set up in the database before they can be used.  This can be found under Administration; Module Options, and then click the Special Deductions button.

 

Then, back to the employees page, click on the Deductions Button.  Fill out per sample below.  Again, this is for EACH EMPLOYEE.  Any additional deductions not in our Chart Of Accounts must be added there first.  Click Ok when complete.

 

QTD / YTD Button

If you do not start the Payroll on January 1, then you must enter the previous quarters earnings.  It is recommended that you start the payroll at the beginning of a quarter as you will have the earnings already calculated manually.  Click on the QTD / YTD Button.  Fill in the form as shown.

  •     Deduction 1 = FICA
  •     Deduction 2 = Medicare
  •     Deduction 3 = Federal Withholding
  •     Deduction 4 = State Withholding
  •     Deduction 5 = Any City Tax
