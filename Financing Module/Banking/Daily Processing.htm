Banking Processing



Note: Always use Arial fonts on all reports.

Main Banking Help can be found in the Banking Module.

When the Banking Module is completely set up, all banking deposits made from the Cash Drawer Entries, are posted directly to the Banking Module.  Likewise, all Payroll and Accounts Payable checks are automatically entered into the Banking Module.  There is no daily entries necessary except to do the bank reconciliation when it arrives.  The bank reconciliation is done in the Banking Module under Processing, Reconcile Accounts.
