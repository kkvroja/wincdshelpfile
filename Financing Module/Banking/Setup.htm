Setup



Open the Banking module for the proper store by entering L1 for Location 1, L2 for Location 2, etc.  Enter the User ID and password  (You must call Custom Design Software to get this code). 

When you are ready to start this module, wait until you have your monthly bank statement in balance.  Make a list of all outstanding checks that have not cleared the bank and also any deposits In Transit which have not cleared.  Take the banks balance and add to it the total amount of the outstanding checks.  Subtract from this figure any outstanding deposits which have not cleared.  Use this final figure and enter it into the banking module as the beginning balance.  This is assuming the the A/P module has been running and the checks not cleared are written from the A/P Module. 

Enter the Banking Module, click on Administration and fill out your banks information as indicated.  Account number requested is 10200.

Next, click on Processing, Account Register. Click on Enter Adjustment.  For reference field use Beginning Balance. Then enter the amount from your recap in amount field.  General Ledger account would be 10200. Date should be set back to statement date.  Click Ok to post the transaction.  The Banking module is now set up.

Finally, the banking deposit from the Cash Drawer Entries must be set so that the bank deposits go automatically into the Bank Manager.  To set this up, from the main menu, go into File, Store Set Up, Miscellaneous.  There you will see on the left options list, where you can check off Post To Bank Manager.  Check this option, click the apply button and return to the main menu. This option will take effect the Next time you open the software.
