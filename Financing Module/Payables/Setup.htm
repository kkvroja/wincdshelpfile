Setup



Open the Payables module for the proper store by entering L1 for Location 1,  L2 for Location 2, etc.  Enter the User ID and password  (You must call Custom Design Software to get this code). 

Click on Administration, Vendor Maintenance, click Add.  Select an abbreviation for the vendor which will be used as a quick Look Up for that vendor.  It can be up to 8 characters.  Do NOT use the inventory 4 digit code.  It is for a different purpose.  Fill out the addresses on both sides as requested. You can copy and paste if they are identical.

Physical Address is the address of the factory. This must be filled out for the PO module to pull the addresses/telephone numbers from the database.

 If you are paying a factor, enter the name as Bassett Furniture Company or First Factors.  The Remittance Address is printed on on the check

Physical Address is the address of the factory. This must be filled out for the PO module to pull the addresses/telephone numbers from the database.

If you get any cash discount, click on payment terms and enter the discount rate and days. 

Finally, you need to click on General Ledger Account and enter, from your Chart Of Accounts list, pick the proper Expense Account.  For example, 11500 is for furniture invoices and 50500 is for freight. The Payables Account and the Cash Account will already contain the proper value.  Instead of looking at the Chart of Accounts, you can hold down the shift key and then mouse click into the expense field.  The Chart of Accounts will pop up automatically.

 After you complete each vendor, click on UP DATE Button to save the vendor.  After all vendors are entered click on OK to exit. 

Hint on Name:  Syncing the Purchase Order Module to Accounts Payable must be accomplished in a certain way.  If done properly, the complete address will be visible on the PO instead of just the manufacturers name.  Here is the trick.  In Inventory, the name of the company is a max of 16 characters.  However, the Name of the company can be around 50 characters.  What the software does is look at the Inventory name, count the characters is uses an then goes down the AP list of Vendor names to find a exact match.  If it finds the match, then it shows up the complete address on the PO’s.  For example; in AP, take the name Bassett Furniture Company.  But in Inventory, use commonly used the name Bassett.  It knows this is a match as there are 7 characters in Bassett.  As it goes down the AP vendor list of names, using only 7 characters, it finds an exact match; Bassett Furniture Company.  So the rule is, if the number of characters in the inventory name match exactly as many characters in AP, it’s a match.  But, in Inventory if you happened to use the name Bassett Furn., it tries to match the first 13 characters in Bassett Furn. including the period, so it doesn’t match! 

The system is now ready to accept invoices directly or from the Purchase Order Module.
