Accounts Payable Processing



Note: Always use Arial fonts on all reports.

Main Accounts Payable Help can be found in the Accounts Payable Module.

Before you begin entering invoices and credits, make sure all new vendors are entered into the system.

  1. Open the Accounts Payable module.
  2. Click the Process Button.
  3. Click Enter Invoices and Credits.
  4. Click the Add button.
  5. Drop the Vendor Code dropdown box, and find the vendor.
  6. Enter the Invoice Number.
  7. Set the Invoice Date, depending on how you pay your bills (Receipt of goods or Invoice Date). All terms will be set from this date.
  8. Enter the Invoice Amount.
  9. If freight is added to the invoice, enter, on the right side of the section, the distributed the freight.
    Add account #50500 for freight.
  10. Click Add.
  11. Repeat for each open invoice. Click Stop Add when done.
  12. Click the OK button to return to the main menu.
  13. Click Reports.
  14. Click Open Invoice List.
    This will print a list of all open invoices in the system, sorted first by discount, then by due date.
  15. On the main menu, click the Processing Button.
  16. Click Select Invoices for Payment.
  17. Select the vendor you want to pay in the dropdown list.
  18. From the open invoices on the screen, you can pay everything by clicking the Pay All button, or by individually clicking on each invoice. Total Cash Required is a sum of all the listed invoices.
  19. Continue to use the dropdown to select more invoices to pay.
  20. When done, click the OK button to return to the main menu.
  21. Click Reports.
  22. Click Select to Pay List.
    Use this list to make sure everything you want paid gets paid.
  23. Remember to use Arial font for all lists.
  24. Click Print.
    Make an Accounts Payable backup
  25. Click Reports.
  26. Click Print Checks.
  27. Click Processing.
  28. Click Finalize Payments.
