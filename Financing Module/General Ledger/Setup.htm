Setup



Note: Always use Arial fonts on all reports.

Open the General Ledger module for the proper store by entering L1 for Location 1, L2 for Location 2, etc.  Enter the User ID and password  (You must call Custom Design Software to get this code). 

  1. Open the General Ledger module.

  2. Click Reports.

  3. Click Chart of Accounts List, choose Arial font, and click Print the Chart Of Accounts.  Do not delete any accounts but you can add additional accounts if needed.  You can change the names of the accounts if you feel it necessary. 

Custom Design Software's General Ledger is used for Month End financials only!  All of the basic modules have their own reports and self balance.  When they get imported to the G/L at the end of the month, they will already be in balance. 

Depending on what month you start the G/L you first must set that month in the software.  Click on Administration, Module Options.  Set the Current Number to the month you are starting.  Notice that the date range is now set for that month.  Click OK and exit from the G/L menu. 

First you must enter your last Balance Sheet in the current period.  To do this, click on Processing, G/L Transactions.  Select the A/P Journal from the drop down list.  Click Add and follow the help menu in the G/L Module.  After your complete balance sheet and expenses have been entered, from the Reports Menu, print out the Balance Sheet, Profit and Loss Statement, and Trial Balance.  These should exactly match your last manual financials.  If not, you must correct the errors before you continue. 

Make a G/L backup before you continue.  The G/L is now set up and ready for the current month.
