General ledger Processing



Note: Always use Arial fonts on all reports.

Main General Ledger Help can be found in the General Ledger  Module.

Custom Design Software's General Ledger is used for Month End financials only!  First, make a current Back Up of the G/L.  Make sure you are in the correct accounting period.  Click on Administration, Module Operations.   Double check the Current Period number.  The actual date range is shown on the screen.  If it is incorrect, change the period, which will change the date range. 

Next, you are ready to import the transactions.  Click on Processing, Import Transactions.  All modules import separately.  Do them one at a time.  The G/L is set to import Report Only. This gives you the opportunity to verify transactions before you  import the data.  If the report is correct, Import with the option set for Transfer and Report. Do the same for each module.
