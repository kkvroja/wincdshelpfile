QuickBooks Interface

WinCDS has the ability to interface with the QuickBooks accounting package by Intuit.

The software will work with QuickBooks 2004 and beyond (it may also work with some previous versions).

To use the QuickBooks interface from WinCDS, it must first be selected in the Options section of the Store Setup.

Once the QuickBooks option is selected, you must then set up your account mapping information off of the QB account setup page. Select General Ledger from the Main Menu and select the QuickBooks Interface Option.

Integration points

While sales, inventory, and the like will still be tracked by WinCDS, WinCDS provides three key areas of integration:

  • Bank Deposits
  • Purchase Order Billing
  • Month End Transfer
