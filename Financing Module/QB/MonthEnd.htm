Month End Transfer - QuickBooks Interface

At the end of each month, you should run the Month End Transfer. It will scan the Gross Margin data and export summary information to the accounts you set up. This will reflect the data that is available daily on your Daily Audit Report. Simply select the month & year you want and click the button. If you re-import the same date range, you will be prompted if you want to re-import the data, only import new data, or cancel altogether.
IMPORTANT: If you re-import existing data, there will probably be duplicate entries in your accounting data and your figures could end up incorrect!
