Add New Credit Apps

From the Installment menu, click on Credit Applications | Add new applications.

Fill out the application as the fields indicate.
NOTE: You must fill out the Date of Birth (DOB) and Social Security Number (SSN), which are high-lighted.




Click Save to save this form, Cancel to exit without saving.

You can also print this form by clicking the Print button.

If you want to obtain an instant credit report from Equifax, click the Equifax button.
