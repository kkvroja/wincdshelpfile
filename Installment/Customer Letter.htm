Customer Letter

To access Customer Letter, go to Installment: Edit Accounts. This feature allows you to easily print an overdue letter to a customer.

To compose a customer letter, click Start-Programs-Accessories-WordPad. Select File-Open, navigate to the C:\InventPX\ folder, and select Rich Text Format (*.rtf) under Files of type. When you are done composing the letter, click File-Save As, and enter a file name for this new document. After it is saved, you may exit and return to WinCDS.

For example, instead of entering the customer's name manually each time, simply enter %07 %08 on the Customer Letter.  WinCDS will automatically replace %07 %08 with the customer's first and last name, etc.
