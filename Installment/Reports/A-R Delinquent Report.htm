A/R Delinquent Report

To run the Delinquent Report click Installment, Reports, Delinquent Report. A "Delinquency Reports" form will appear. Choose to sort either by Name or by Account Number, then click Print or Print Preview.

The A/R Delinquent Report will show any open contracts with accounts which are over 30 days delinquent.  It's format is the same as Ageing Accounts.
