Void Sale

The Void sale in F1 explains that if a customer wants to return an order, use Void to cancel the sale.
In actual, we can void a sale only if the customer wants to return an entire order or we can void the sale if we want to close the entire sale.
So, if the help centre clearly states that if the customer wants to return the entire order that’s when we void the sale, that would be more clear and understandable.
To void a sale that has been written, click Void Sale on the Order Menu.

Selecting a Sale

Enter the Bill of Sale number, then press Enter. The following Void form will show on top of the Bill of Sale.



Note: The date is especially important at the end of the month. Be sure the void is posted in the correct month.

You should always use the default date, unless you are voiding a sale that took place before you balanced at the end of last month.

Void Options

You have several options on the Void Screen.
  • Void the sale and return the money as paid.
  • Some times, the customer paid by cash and you have to write a company check.
  • You want to issue a Store Credit.
  • You are keeping the deposit and writing off the sale.


In either case, click on the option for this sale.

Company Check

If you choose Company Check of Store Credit, you will be able to do all or partial refund or credit.

Store Credit

If you choose Store Credit, the software will print out a Store Credit Memo for the customer. It will generate the Bill Of Sale number with an SC added for Store Credit to the number.

Note: Once a store credit has been issued to the customer, it can be applied to any sale in the future. To use all the Store Credit, simply Void out the Store Credit and apply the full amount of the Store Credit to the new sale under Customer Payment. If you only want to use part of the Store Credit and leave a balance remaining, bring the sale up under Customer Payment, Make a MINUS payment (-XXX.00) for the amount of money you want to use on the Store Credit.

Forfeit Deposit

If you choose Forfeit Deposit, you need a password to proceed. If the password is accepted, then the sale is Voided, the money is forfeited to income and the sale is locked.

After Voiding

After you click on the Ok button, the Void Sale is processed.
  • Any inventory is returned to stock and any Special Ordered items, the Purchase Orders are automatically Voided.
  • If there are any "Special Specials" that have been received, you will be prompted to add these items into the inventory under New Items.
TIP: If only some of the items need to be cancelled (instead of the whole order), use Customer Adjustments to add/remove individual items, delivery charges, etc. for that sale.
