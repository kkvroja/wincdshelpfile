Store Login

Selecting a Store

If your company has more than one store, you can select your location.

Each time WinCDS starts, the software will assume you are at location 1. If you are at a different location, you must first log into your appropriate location each time you load the software.

Changing Stores

To log in to a different store:
  1. Go to Store Login on the File menu (or press F2 from the Main Menu)
  2. Select the location you want from the list provided (alternately, for convenience, you can press the number corresponding to the location you desire. e.g. Press the number 1 for location one, press 2 for location two, and so on)
  3. Press OK


Note: Inventory is generic between stores. The default location for selling merchandise from Store Locations have been preset to your specifications.
