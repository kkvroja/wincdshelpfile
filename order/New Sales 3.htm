Style or Carpet - Entering a Style Number

If you have selected either Enter Style Number or Carpet, you will be prompted for a style number.



You can either type in a full style number, a partial style number and select from the list on the right, or click on either Vendor Name or Description for alternative searching.

What if the style number doesn't exist?

If you enter a style number that does not exist, you will be prompted with the following dialog box.

From this screen, you can do one of the following:
  Re Enter style number Re-enter the style number if you entered it incorrectly
Enter Item Not In Inventory Enter an item not in inventory. This item will show up with the FND status. This can be used for items you find on your floor that don't have any particular place in your inventory status but which aren't being special ordered.
S/O Item Not Carried Enter a Special Order item that is not in your inventory list. This creates a Special/Special sale which will show up with a status of SS.
S/O Lay-A-Way Not Carried Enter a Special/Special item for Lay-a-way (SSLAW). This is a combination of the S/S feature above and the Lay-A-Way option.


Choosing the status and quantity for your item(s)

  • Finally, you must choose the quantity, stock location, and status of each item sold.  To enter a quantity, simply enter the number in the Quan box (take note of the amount available which is displayed to the right of the quantity box).

  • Also, select the store from which you want to pull this this stock from by selecting the appropriate location option button.

  • The last selection you must make is that status for the item. Select one of the five options from the bottom of the form. The options are defined below:



    •   Reduce Stock (default) Holds the item indefinitely for later delivery or pick up. This reduces the Available For Sale as the merchandise is being held.
    Take With Item Use this for cash-and-carry items. This will reduce inventory and deliver the item in one operation.
    Tag Incomming Stock If this item is out of stock, but there is some on order, this will reserve the item for the customer when it arrives. When the purchase order shipment is received, it will automatically find all orders tagged against it, mark those orders received, and show them on the receiving report. Any available stock left over from that purchase order will be automatically added to inventory, and you will be prompted to print out a new price tag.
    Special Order Choose this if there are none on hand, and none on order. You can also change the complete description, which follows through to the purchase order module. This could be a different fabric or color.
    Lay-A-Way This is used generate a quote. Lay-A-Way does not reduce the stock. After the customer confirms the Lay-A-Way order, the item can be converted to either a stock or special order very easily. Once the customer confirms the order, just double clik on the item's Status column which is LAW and you will be prompted to turn this into a stock or Special Order.
