Credit Sales Report



The Credit Sales report shows all sales sent to an outside finance company.   Theses are also part of your Receivables but are separated from the Back Order Receivables as the money is coming from different sources. 

This report is perpetual so the date is just the date you want to read on the report.   Click on how you want to sort the report by Name or by Sale number.  Sale number is preferred.  You can also select If you want the report separate by sales person, by ageing, or by sale date. 

Finally, choose either Print or Print Preview. Print will print the report on the default printer, Print Preview will show the report on screen. For help with the Print Preview function, see the Print Preview help page. 

This report is mandatory to save at the end on the month.
