Advertising Report

Assuming you have set up your default Advertising Types in the Store Setup, and either have your sales staff select Advertising Type in the new sale drop down or are forced to select an Advertising Type in the drop down, this is what activates this report.  The question is "How did the customer come into the store".

Note:  In the Store Setup, you can set an option to force an advertising selection.

           

First, Set the date for the time period you want statistics on. Next, if you have multiple stores, select the store you want the report for in the drop down.  Checking the first option gives a more condensed report because the cities can be spelled out in many ways; i.e., Michigan, Mi, Mich.  etc.  So checking the first option combines all cities with the same zip code alphabetically.   Option 2, Sort report by Zip Code gives your list in zip code order. 



Each page of the report is a different advertising type.
