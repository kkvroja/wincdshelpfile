Inventory Scanner Function Keys

From anywhere in the software, you can also use the special Function Keys. To use these, press the FN/Alpha key located at the bottom left of the keypad until you see an F in the bottom right of the screen above the the batter indicator. This will indicate that the keypad is in Funtion Key mode.
Now, instead of the keys entering characters, each number on the keypad will execute the following operation:

Available Functions

  1. Toggles between Normal and Silent Mode. In silent mode, the scanner will not beep at all, neither for confirmation or for errors. Be sure to watch the status screen when entering data.
  2. Starts/Stops the scanner's networking features. Switching the networking features off will provide a longer battery life. Note that the network will restart automatically whenever any feature requiring it is activated.
