Remote Setup - Utilities - Inventory Scanner

From this screen, you can configure the scanner to connect to WinCDS. Enter the IP address of the server, the port (usually 3000 unless you changed it under the Configure Wireless Scanner feature under the file menu of WinCDS. You can also specify a password if one was specified in WinCDS for the wireless scanners. Finally, you can test the connection or stop the network if desired.
