Stock Location - Warehouse Tracking - Inventory Scanner

Use this to transfer stock from one location to another.

Stk Location

You will be prompted for the following fields:
  • Style
  • Old Bin
  • New Bin
  • Serial
Style must be a valid Style number and New Bin must be a valid location barcode. Old location can be left blank, but must not match the new location. Serial Number can also be left blank, but if provided, it must be found in the database already. Serial Number and Old Bin should not both be left blank because no specific item would be identified for transfer. If you would simply like to add an item to a loction, use Stock Setup.
