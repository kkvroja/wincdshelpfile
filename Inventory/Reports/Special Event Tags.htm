Special Event Price Tags

Overview

If you want to run a special sale for Mothers Day, Fathers Day etc., the Special Event Price Tags will allow you to create an Over Tag to place on top of your regular tag.

Tag Stock

To create the tags, we suggest using blank tags from Sale Signs (http://www.shopsalesigns.com/). Their number is 800-346-8116.

Creating Tags

Once you have the blank signs, Select the Size of tag, either Large or Medium. You then can select whether you want tags for Everything in the store or print tags by Department or Manufacturer. Select the Gross Margin to have the computer calculate the prices. Finally, you can change the Font and Size of the prices. Load the Laser Printer with blank tags and you are ready to print.
