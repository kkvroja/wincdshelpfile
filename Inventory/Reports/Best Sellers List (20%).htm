Best Sellers List

To print or view the Best Sellers List, click Inventory: Reports: Best Sellers List (20%). An "Inventory Best Sellers" form will appear. Enter the Report Date, choose the sort options, then choose either Print or Print Preview. Print (default) will print the report on the default printer, Print Preview will show the report on screen. For help with the Print Preview function, see the Print Preview help page.

The Best Sellers List shows the top 20% best selling merchandise.
