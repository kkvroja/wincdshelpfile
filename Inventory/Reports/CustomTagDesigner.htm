Saving & Loading

You can load and save your custom design tags quick and easy. There are two types of tags to load:
  1. Basic Tag Templates - These are the basic system layouts that can help get you a start towards a tag design.
  2. Previously Saved Tags - These are saved layout tags that were either provided with your system or that you have created.
WARNING: Please remember that whenever you load a new tag or exit the screen, all work the current tag you may have been creating will be lost. If you wish to keep the tag you're working on, click on Save Layout.

Loading a Basic Template

To load a basic template, select one of the basic system layouts from the box that loads with (Default). The current tag will be cleared and will be loaded with the system layout you select.
NOTE: The basic templates cannot be changed through the software.

Loading a Saved Tag

To load a previously saved template, simply select from the drop-down list labeled Layout Name. The tag you select will be loaded into the display window.

Saving the Tag Layout

If you are satisfied with a layout you have created, or simply want to save the progress you have made so far in laying out a tag, click on Save Layout. You will be asked what you want to name this layout. Be sure to type something that will be recognizable later, as you may end up with many later (e.g. '2007-Christmas-Sale'). Click OK to save, or Cancel to cancel saving.

Customizing Your Tag

You can customize just about everything about your tags. See below for the options available.

Setting the Tag Dimensions

SizeDimensions
Small1" x 2"
Medium3" x 4"
Large8.5" x 11"
CustomAny (Original Default is 3" x 3")
The first thing you should decide is what the dimensions of your new tag should be. There are three preset tag sizes: Small, Medium, and Large.
Additionally, you can make a tag any size you want using the Custom tag size. After selecting this size, enter the width and height in inches into the boxes below the size.

Custom Tag Sizes

To select a custom tag size, select Custom from the Dimensions drop down. Then, enter the dimensions of your custom tag (in inches) in the boxes that appear. The display region will update to the dimensions that you specify.

Selecting your Currency Format

In the box at the top is a checkbox which will determine the format that the currencies on your tags will be printed. Selecting the box will change the format of the prices to include a dollar sign. Leaving it blank will print prices without this formatting.

Selecting the Fields

To select a field, simply move your mouse over the field. The field should turn green.
Additionally, you can click on the up-down arrows beside the Field Name property to select the field you wish to customize.

To choose whether a field should be displayed or not, check or clear the check box beside the field in the Tag Items list box. You can also click Select All or Clear All for convenience.

Extra Fields

If you wish to add additional text, such as captions for the prices, or sale information, or for any additional items you wish to add to your price tag beyond the default data tags, you can click the Add Text button. An additional field will be created with a Caption property that you can edit to display the text you want.

You can also remove extra text items that you have created using the Rem Text button.

Pictures

You can also add pictures to your tags. To add a picture, you start by adding a text field.
Then, select that this extra field is a picture using the selector in the lower right. Then, where the caption would go for a text field, simply enter the filename.

TIP: If you are creating a background-image that will have text on top of it, make sure that it is the LAST element (i.e. make sure that it is the last extra text field that you create). This will ensure that the text will print on top of the image instead of invisibly printing behind it. Also, there will be no 'white box' around the text. It will be printed right on top of the image.

Image Dimensions

To specify the dimensions for your image, simply enter them into the boxes below the file name. If you want to lock the aspect ratio of the image (that is, you don't want it to be stretched and skewed out of its original proportions), simply click the Lock check box beside the image dimensions.
TIPS:
  • You don't need to enter the full filename. If you place your images in the WinCDS image directory (C:\InventPX\ on the server or I:\InventPX\ on a workstation), you only need to enter the name of file, without extension, and it will be detected. This can be a great time-saver and will also help you keep all of your WinCDS images in one place. WinCDS also will accept a full filename.
  • If you want to display the picture you have previously stored in your computer for the inventory item, enter the pound sign (#) in the Filename box. A temporary place holder (a large red box) will temporarily be displayed, but when you go to print your tag (or preview), the image that shows up in View Stock Item will be displayed instead. Be sure to lock the aspect ratio for your image so that it shows up how you expect!

Arranging and Customizing the Fields

Once you have selected the fields you wish to have displayed on your tag, the next step is to arrange and adjust them.

Positioning the Fields

To move a field, simply drag it to the position you want. To drag an item, position the mouse over it, click and hold the left mouse button. Move the mouse to the new location. When you have reached the place you wish to place your tag, let up the left mouse button. For your reference, the location of your mouse pointer or the object you are currently positioning is displayed in a blue box below the properties.

Field Properties

You will also need to set the properties of your fields. See below for the options available.

Field Name

This displays the field you are currently editing. You cannot change the name of the field. To select a different field, either move your mouse over the top of it (the field you want to find will turn green), or select the field using the up and down arrows beside the field name.

Font Name

Use this drop down to select the font that will be used to display the text. The Default is Arial but use this to create just the look you want.

Font Size

Change this value to modify the size of the text displayed. For the barcode, be sure to keep this large enough so that your scanner can still read the bars.

Font Color

If you are printing on a color printer, you can change the color of any of your items.

Caption

For the extra text fields, you can modify the caption of your item.

Character Style


Select one or more of these options to change how your text will be displayed.

Character Alignment


Select how you wish this item to be positioned. Note that when viewing things such as descriptions or style number, the lengths of items for individual stock items may be very different. You can use the Character Alignment property to align the text depending on your tastes.
Alignment TypeDescription
Align to Top LeftThe fixed point will be the top left of the text.
Align to Absolute LeftThe text will always be at the absolute left of the tag.
Align to CenterThe text will always be centered in the tag.
Align to Absolute RightThe text will always be at the absolute left of the tag.
Align to Top RightThe fixed point will be the top right of the text.

Field Type


If you are editing an extra field, you can also select whether this field will be an image or text.

Multiple Styles on a Tag

Sometimes, it is beneficial to have multiple style numbers displayed on a tag, such as a mattress store listing the prices for Twin, Full, Queen, and King sized mattresses. It is also sometimes useful to have multiple groups of styles, such as if you sell a mattress and a box spring separately. This can be accomplished by the Multiple Styles box.

While they will not be displayed by the default tag fields, they can be accessed by adding extra fields.

Entering Multiple Styles

To enter mutliple styles that will be displayed on tag, simply enter each style on its own line in the multiple styles line. You can have as many styles as you want.

To have style groups, put each group on its own line, with each item in the group separated by commas.

Referencing Multiple Styles

Once you have entered your extra style numbers, you need to create the extra fields to display them (or, as a short-cut, you can select one of the already created layouts or templates to display them). To display a field from one of the extra styles, you must enter one of the available codes into the box, as described below.

All of the codes start with the character # (pound sign). After that, the field name is specified (see table). Then, a colon, and finally, a designator specifying the line and, if applicable, the position in the group.

For the following examples, assume the the multiple styles box has the following four lines:

    6000-066,6000-091,6000-085
    557
    3000,3001
    44444

 Some example codes would be:

Grouping Style Items

To have groups of items, simply put multiple styles on the same line, separated by commas. You can have up to 10 items in a group.

Mattress Tags


If you want to create a tag for a mattress group, double click on the Multiple Styles box, and a form will pop up to help you enter all the styles. In each of the boxes available, you can double click and look up a style number. Either click OK or Cancel when you are done to save or cancel your entries. The values will be filled in on the Tag Designer.

Printing your tags

If you are ready to print a tag, select the Print Tag button. It is always a good idea to save a tag before printing with it. In addition, enter any style number that you wish to display on this tag.

Preview Style

To see how a particular style will be displayed on your tag, enter the style number into the Style box. Press TAB to leave the field and the tag will be updated to display the tag. Leave this field blank to view placeholders instead of actual data.

Printing a single tag

To print the currently previewed style, click Print 1 Tag. The currently previewed tag will be printed.

Printing Multiple Tags

To print multiple tags, select the Manufacturer, Department, or partial style numbers you wish to print. Select whether you wish to print one for each On Hand item or simply 1 tag per style number in your selection. The total number of tags that will be printed will be displayed in the box labeled Tags to be printed:. When your selection is complete, click Print Tags to print them.

Returning to the Main Menu

To return to the main menu, be sure you have saved any work on tags you wish to preserve, and click Menu.
