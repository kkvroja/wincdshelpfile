Bar Code Data Collector

This section is for taking Physical Inventory, not loading original inventory.

NOTE: Do not use this feature for loading the original inventory. Use the Load Original Inventory feature in the File Menu.

Using the CipherLAB Data Collector

  • Turn on the CipherLAB by pressing the button on the lower right corner of the unit.
  • To navigate the menus, either press the number corresponding to the item you wish to activate, or use the up/down arrows to highlight it and press one of the two blue buttons (like the Enter key on the keyboard).
  • If you are not already at the main menu, press the ESC key until you are.
  • To take your physical inventory, select menu option 3 (either pressing 3 or or pressing down twice and pressing one of the blue Enter keys).
  • Select Scan (press Enter on menu option 1).
  • Read the warning message and Press Enter (or ESC to cancel).
  • Enter your location (1-16).
  • Aim the CipherLAB at the barcode at the barcode and press the large yellow button on the top of the scanner for each item you scan.
    • If you have duplicate items, you can scan each item or scan the first item and enter the quantity on the keypad, press the enter key to accept.
    • NOTE: Do not scan kits.
  • After the last item is scanned, be sure to press the Enter Key to accept the last scanned item.
  • When done, press the ESC key to return to the Phys. Inventory menu.
  • You can use multiple data collectors.
  • If you want to take Inventory at another location, you can select another location from the Phys. Inventory menu and begin scanning the Inventory at that location. The CipherLAB will hold all stores inventory.

Processing the Data From The Data Collector

  • To upload the data from the CipherLAB, at the Phys. Inventory menu, select Upload Data (Menu option 2).
  • Select via Base (in which case the CipherLAB must be properly seated in its cradle after selecting this option) or via Wireless.
  • From Custom Design Software's Main menu, press Inventory | Reports | Barcode Data Collector. The form below will appear.


  • From the form, click on Import Data. Select the 8061-C option (either via Base or via Wireless) in the drop down. Press Import for the Import to begin.
  • When you are prompted File Successfully Written, click the Ok and then the Close.
  • Remove the CipherLAB from the cradle and press the Escape key.
  • Be sure to shut off the CipherLAB by pressing the button on the lower right corner of the unit. Keep holding until the unit turns off.


  • Special Note:You can import multiple times from multiple scanners in Step #1. Just continue to upload the data until all your scans are uploaded onto the computer. When you are ready to combine and use all of the scans, then proceed to Step #2.

  • From the form, click on Compare Inventory. This compares the scanned items with the actual Inventory in the Database including Special Orders Received, and merchandise being held for customers.
  • The final step is to select how you want the reports to be printed. The Default settings are set to print a Discrepancy Report on everything. If you want individualized reports, you can report by Department or by Manufacturer for all stores or each store individually.
    • If you select Department, click on the Department you want to report on.
    • If you select Manufacturer, click on the Manufacturer you want to report on. 
    • If you want individual stores, click Stores and check on the stores you have scanned.
Finally, select Print on Screen or Paper.

Click Generate Report to generate a Discrepancy Report based on the report settings.
