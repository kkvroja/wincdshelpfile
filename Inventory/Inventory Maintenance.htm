Inventory Maintenance



Changing Item Contents

To change the contents or pricing of an item, click Inventory Maintenance on the Inventory Menu.

The "Inventory" form will show information about the style number you choose and, rather that simply viewing the item, will allow you to change edit the fields, including prices.

Once you are satisfied with your changes, click Apply to save them.

Inventory Modes



There are three inventory modes to choose from.
  • Manual - Allows you to enter the style number you choose
  • Automatic - Allows you to pick a manufacturer and go through each item with the Next and Back buttons.
  • Excel - Allows you to change prices using an Excel (XSL, CSV, or other Excel compatible file extension).


Note: Using Back/Next will not save the record. To save the record, click Apply before Back/Next.

Excel Import

The Excel feature brings up the following form. To use this feature, follow the following simple steps.



  1. Browse to the filename you want and click Load.
  2. Once the file is loaded, enter the worksheet number and configure the columns for Style, Sale Price, and List Price.
  3. Review the changes before committing.
  4. Click OK to import the price changes by style.

Terms Used

M/STOCK
It is the stock limit value after which the store must make a purchase order.
E.g if the stock reaches a value of 10 the system will automatically generate a purchase order or remind the owner to create a purchase order.


PO BUTTON
This button is used to show all the stock items that have reached the M/STOCK level.



COST
It is the cost at which the manufacturer sells the furniture. e.g 100$.

FREIGHT PERCENTAGE/ AMOUNT
It is the percentage or amount at which the manufacturer of the furniture ships the furniture to the store address. e.g.40%.

LANDED COST
It is the cost at which the stock lands in the the store.
I.e. (cost + freight percentage/amount = landed cost). e.g taken cost as 100$ and freight % as 40%, thus landed cost will be 100$ +40% i.e 180$= 280$.


G/M%(GROSS/ MARGIN PERCENTAGE)
It is the margin higher cost at which the store sells the stock. e.g 60%
