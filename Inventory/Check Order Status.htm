Check Order Status

To check the status of an order, set up Delivery, or Pick Up, click on Check Order Status from the Inventory Menu.



Check Order Status

Enter the Sale Number, and press tab. The order information will appear as shown.

Editing Status

If you need to change an items Description like a Special Order where the fabric is changed, type over the existing Description and click Apply. The new Description will transfer to the Sales Order and the Purchase Order where you can do a reprint.

You can change or add the Cost and Freight. Click Apply to save all changes.

Delivery Dates

To set up delivery or pick up, first you might want to check the Delivery schedule. Click the Delivery Calendar button. The Calendar will generate and you can find the correct delivery day.

If the customer wants to set up delivery or pick up, check the Delivery or Pick Up checkbox, and set the date using the built in calendar.
Note: You must set the delivery/pickup date for each piece or use the Set All Items for Delivery check box.s

Special/Special Department

If the item is a SS (Special Special; Catalog Sale), it would not show the department for that item. Highlight the line item that needs a department and from the Drop Down list select the correct department for each SS.

Note: You can also change the Cost, Freight and Department while receiving the sale or when you are delivering the sale.
