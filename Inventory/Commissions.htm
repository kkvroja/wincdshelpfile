Commissions

To pay commissions or edit Sales Staff for a sale, click Commissions from the Inventory Menu.

Setting up Commissions

The Commisions must first be set up within the Store Setup Commissions Section. From there, you will be able to select payment on Delivered, Written, and select from various commissions formulas. Additionally, customization options are available by contating Custom Design Software.

IMPORTANT: Please be sure to set up your commissions in the Store Setup before running Commissions for the first time. You can contact WinCDS for more details on this one-time setup.

Running Commissions On Your Database

The Commissions process consists of two steps.
  1. First, the comissions reports are generated and either printed to the screen or the printer. At this point, nothing has changed in the database.
  2. Second, you can choose to post the Commissions to the database. This actually marks the sales commissions as paid based on the reports generated.

Step 1 - Generating Commissions Reports

To generate the report, select the options from the top of the form.
  • To preview the report, instead of printing it, select the Preview checkbox.
  • If you want to view individual sale sub-totals, select the Show Sale Sub totals checkbox.
Once you have selected the generating options you want, click the Recalc button.

You have now generated your report (either on screen or printed), but have not posted the changes to your database.

Step 2 - Post Commissions

When you have simply run the reports, the form will display NOT POSTED to indicate that the commissions have been run but not posted to the database.

After you have reviewed the reports, made any and all Sales Staff adjustments and run the reports again, and are satisfied, click Post to make the changes permanent.

Additional Operations

In addition to simply running Commissions, you also have several operations you can perform on your data that will affect your commissions.

Edit Salesperson



If you need to adjust the salesperson on a sale, check the Edit Salesperson checkbox.
Enter the sale number you wish to adjust and select the sales person and the sales percentages for the sale. You will need to re-run the commissions report before posting after you do this.

Edit Spiff



If you need to adjust the spiff on an item, check the Edit Spiff checkbox.
Enter the sale number you wish to adjust, go to the line-item you wish to change the spiff on and simply change it. You will need to re-run the commissions report before posting after you do this.

Reprinting Comissions Reports



To reprint a prior period already posted, check the Reprint Report checkbox. Select the date the desired report was run from the drop-down list and run the report. The selected report should print to the printer or screen, depending on the option you selected.

Additionally, the PDF button allows you access to the Report PDF archive. If enabled, every time you run the Commissions report, a PDF version of the report will be generated for a perpetual archive.

You can either view the archived report, or generate a new commissions summary for the date based on the database tracking.
